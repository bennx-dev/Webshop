Dit is een fullstack mobile-first leerproject waarin ik experimenteer met webdevelopment en architectuurkeuzes.
De focus ligt op het begrijpen van hoe frontend en backend samenwerken, met nadruk op backend-logica.

De frontend is bewust klein en licht gehouden.

## Tech stack
- Backend: Spring MVC, Spring Data
- Frontend: React, HTML, CSS, JavaScript
- API: REST API

## Doel
- Inzicht krijgen in fullstack development
- Werken met API’s
- Backend-gedreven applicaties bouwen
- Minimalistische frontend toepassen
